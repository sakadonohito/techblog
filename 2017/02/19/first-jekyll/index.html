Jekyllを試す

Ruby製の静的サイトジェネレーターを試してる。

インストール

以下下記URLを参考にいろいろ試したけど、情報が古くコマンドが変わってた。 参考

gem install jekyll
mkdir _site _posts _site assets
mkdir -p assets/css

構造はこんな感じ

├── _config.yml
├── _layouts
│   ├── layout.html
│   └── post.html
├── _posts
├── _site
├── assets
│   └── css
└── index.md

jekyll serve --watch

で編集内容が即時される。

こんな感じでやっていこうと思います。

レイアウトがとてもダサイ感じなので、テーマをちゃんと作っていきたい。

Ruby製だから？ヘルプが分かりやすい。

静的ファイルを生成したい場合はjekyll buildで_siteディレクトリに生成される。
